### ها كيفاش كيكون الكود
```
Variables a,b,res ENTIER 
Debut 
Ecrire("saisie a") ;
Lire(a)
Ecrire("saisie b ") ;
Lire(b)
Res ← a +b
Ecrire("a + b = " & res) ;
Res ← a - b
Ecrire("a - b = " & res) ;
Res ← a * b
Ecrire("a * b = " & res) ;
Res ← a / b
Ecrire("a / b = " & res) ;
FIN
```
